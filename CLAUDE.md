# RC Paper - Informații pentru Claude

## Configurare Proiect

### Compilare Automată
Paper-ul este compilat automat după fiecare modificare în VSCode folosind MiKTeX.
- Nu este nevoie să rulezi manual comenzi de compilare
- Modificările în `.tex` sunt detectate și procesate automat
- PDF-ul este regenerat automat la salvare

## Structura Proiectului

### Fișiere Principale
- `research_paper.tex` - Document principal LaTeX
- `references.bib` - Baza de date bibliografică
- `research_paper.pdf` - Output-ul compilat

### Configurare Document
- Format: A4, două coloane, 11pt
- Limbă: Română
- Compiler: MiKTeX cu pdfLaTeX
